🏥 SevaHealth - NGO Support Portal
Project Overview
A concept-level healthcare support web app designed for NGOs to digitize patient registrations and use automation to prioritize emergency cases in rural health camps.

AI & Automation Idea: "Instant Triage Assistant"
I implemented a Rule-based Automated Triage system using a Python backend. When a patient submits their symptoms, the system automatically scans for critical keywords (e.g., chest pain, breathing issues). It instantly flags the patient as 'URGENT' or 'Normal', helping NGO staff identify life-threatening cases at a glance.

Tech Stack
Backend: Python (Flask Framework)

Frontend: HTML5, Bootstrap 5 (Responsive Design)

Deployment: Hosted on Render (Live Link below)

NGO Use-Case
In high-volume rural health camps, manual sorting of patients leads to delays. This app acts as a Digital Front-Desk that automates the screening process, ensuring critical patients get immediate medical attention.

How to Run
Install dependencies: pip install flask gunicorn

Run the app: python app.py

Open http://127.0.0.1:5000 in your browser.
